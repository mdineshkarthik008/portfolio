# Portfolio Website

This portfolio is now set up as a simple static site with all HTML, CSS, and JavaScript combined into `index.html`.

## Run locally

Open `index.html` in your browser.

## Upload to GitHub

1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Commit and push the project.

## Enable GitHub Pages

1. Open your repository on GitHub.
2. Go to `Settings`.
3. Open `Pages`.
4. Under `Build and deployment`, choose:
   `Source` -> `Deploy from a branch`
5. Select:
   `Branch` -> `main`
   `Folder` -> `/ (root)`
6. Click `Save`.

After GitHub finishes publishing, your portfolio link will usually be:

`https://YOUR-GITHUB-USERNAME.github.io/REPOSITORY-NAME/`

Example:

`https://mdkar.github.io/portfolio/`

If your repository name is exactly:

`YOUR-GITHUB-USERNAME.github.io`

then your link becomes:

`https://YOUR-GITHUB-USERNAME.github.io/`
